# sh-vue-ui

### 启动 
npm run serve
### 打包 
npm run build

### 发布
> 1. git 提交所有内容  
> 2. npm version patch // 升级版本号
    major：升级主版本号 1.0.0 -> 2.0.0  
    minor：升级次版本号 1.0.0 -> 1.1.0  
    patch：升级补丁号 1.0.0 -> 1.0.1  
> 4. npm发布包创建    
    npm run lib
> 5. npm publish 发布
